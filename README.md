 ## 🚀 HR Analytics Dashboard | Tableau, SQL & Excel

Transforming workforce data into meaningful HR insights! 📊

This project is a comprehensive HR Analytics Dashboard built using Tableau, SQL, and Excel, designed to analyze employee attrition, workforce demographics, job satisfaction, and departmental performance.

Through interactive visualizations and KPI-driven reporting, this dashboard enables HR teams and management to make data-driven decisions by identifying attrition patterns, employee satisfaction levels, and workforce trends.

 ### 🔥 Dashboard Highlights

✨ Employee Attrition Analysis & Attrition Rate Tracking

✨ Active Employees & Workforce Distribution Analysis

✨ Department-wise Attrition Insights

✨ Employee Age Group & Gender Analysis

✨ Education Field-wise Attrition Analysis

✨ Job Satisfaction Rating by Role

✨ Interactive Filters for Dynamic Exploration

✨ Executive-Level KPI Cards and Reporting
